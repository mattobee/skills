# Skills

Agent skills for coding agents -- OpenCode, Claude Code, Cursor, and more.

## Available Skills

### designing-agent-teams

Designs, generates, and refines multi-agent coding teams with optimal model-to-role assignments. Produces reasoning documents and ready-to-paste configuration files for OpenCode and Claude Code.

## Installation

```bash
npx skills add mattobee/skills
```

Or install a specific skill:

```bash
npx skills add mattobee/skills --skill designing-agent-teams
```
